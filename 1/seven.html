1.7 Components of Computer System

A computer system is made up of different components that work together to process data.

1. Hardware 🖥️

Physical parts of a computer system.

  • Keyboard, Mouse
  • Monitor, CPU
  • Printer, Scanner

🌍 Example: Laptop, desktop computer parts

2. Software 💾

Programs that run on computer hardware.

  • Operating System (Windows, Linux)
  • Application Software (MS Word, Chrome)

🌍 Example: Mobile apps, desktop software

3. Data 📊

Raw facts and figures processed by computer.

  • Numbers, text, images
  • Unprocessed information

🌍 Example: Student marks before calculation

4. Users 👨‍💻

People who operate computer systems.

  • Students
  • Teachers
  • Office workers

🌍 Example: You using a laptop for study

5. Procedures 📜

Step-by-step instructions to operate system.

  • How to login system
  • How to save files
  • How to backup data

🌍 Example: School computer lab rules
