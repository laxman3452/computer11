1.3 Measurement Units in Computer

Learn CPU speed, frequency, storage units, and how computers measure performance & data.

⚡ Processing Speed

CPU speed is measured in Hertz (Hz).

🧠 What is a Cycle?

A cycle is one complete action performed by the CPU:

  • Fetch instruction
  • Decode instruction
  • Execute instruction

One cycle = one full processing step inside CPU.

📌 Units

  • Hz = 1 cycle per second
  • MHz = 1 million cycles per second
  • GHz = 1 billion cycles per second

💾 Storage Units

Computers store data in binary format (0 and 1).

  • 1 Bit = smallest unit (0 or 1)
  • 1 Byte = 8 bits
  • 1 KB = 1024 Bytes
  • 1 MB = 1024 KB
  • 1 GB = 1024 MB
  • 1 TB = 1024 GB

📡 Frequency in Computers (IMPORTANT)

Frequency means how many times something repeats in 1 second.

🖥️ In Computer System

  • CPU frequency = how many instructions processed per second
  • RAM frequency = how fast data is read/written
  • Network frequency = how many data signals are transmitted per second

📊 Simple Example

If CPU = 3 GHz → It performs 3 billion cycles per second.

More frequency = more data processed per second.
