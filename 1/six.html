1.6 Computer Architecture & Organization

Understanding how a computer system works internally (input → processing → output).

🧠 Von Neumann Model (Basic Architecture)

📥 Input Unit

Accepts data and instructions from user.

Example: Keyboard, Mouse

💾 Memory Unit

Stores data, instructions, and results temporarily.

Example: RAM

🧠 CPU

Main processing unit of computer.

Brain of computer system

📤 Output Unit

Displays processed result to user.

Example: Monitor, Printer

🔄 Block Diagram Flow

Input Unit ➜ Memory Unit ➜ CPU ➜ Output Unit

Explanation:

  • Input unit collects data from user
  • Data is stored in memory temporarily
  • CPU processes data using ALU and CU
  • Output unit shows final result

⚙️ CPU (Central Processing Unit)

➗ ALU (Arithmetic Logic Unit)

  • Performs calculations (+, -, ×, ÷)
  • Performs logical operations (AND, OR, NOT)

🎛️ CU (Control Unit)

  • Controls all operations of CPU
  • Manages data flow between units
