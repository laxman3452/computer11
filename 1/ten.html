1.10 Input and Output Devices

Understanding devices that help in communication between user and computer system.

⌨️ Input Devices

Input devices are hardware components used to enter data and instructions into the computer.

⌨️ Keyboard

Used for typing text and commands.

🖱️ Mouse

Used for pointing and clicking.

📠 Scanner

Converts paper documents into digital form.

🎤 Microphone

Converts voice into digital signals.

🖥️ Output Devices

Output devices display or produce the processed result from the computer.

🖥️ Monitor

Displays visual output on screen.

🖨️ Printer

Produces hard copy on paper.

🔊 Speaker

Outputs sound/audio signals.

📽️ Projector

Displays large visual output on screen.

🔁 Input–Process–Output (IPO) Model

Input ➜ Processing ➜ Output
  • Input: Data entered using input devices
  • Process: CPU processes the data
  • Output: Result shown via output devices
