1.9 Memory in Computer System

Understanding how data is stored, accessed, and managed in computer systems.

🧠 Primary Memory (Main Memory)

Primary memory is the memory that is directly accessed by the CPU. It is fast but has limited storage capacity.

⚡ RAM (Random Access Memory)

  • Volatile memory (data lost when power off)
  • Used for temporary storage
  • Faster access

Example: Running apps, system memory

📀 ROM (Read Only Memory)

  • Non-volatile memory
  • Stores permanent instructions
  • Cannot be easily modified

Example: BIOS system instructions

💾 Secondary Memory

Secondary memory is used for permanent data storage. It is slower but has large storage capacity.

🖴 Hard Disk

Traditional large storage device

⚡ SSD

Faster storage using flash memory

🔌 Pen Drive

Portable storage device

📊 Primary vs Secondary Memory

Feature Primary Memory Secondary Memory
Speed Very Fast Slow
Storage Small Large
Volatility Mostly Volatile Non-Volatile
Usage Temporary processing Permanent storage
