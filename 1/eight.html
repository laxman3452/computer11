1.8 Microprocessor and Bus System

Understanding how CPU processes data and how internal communication happens inside a computer.

🧠 Microprocessor

A microprocessor is the brain of the computer that executes instructions and performs all processing tasks.

➗ Arithmetic Operations

Addition, subtraction, multiplication, division

🧠 Logical Operations

AND, OR, NOT, comparison operations

🎛️ Control Operations

Controls data flow between components

🚍 Bus System

A bus system is a communication pathway that connects different parts of a computer and transfers data between them.

📊 Data Bus

Transfers actual data between CPU, memory, and devices.

📍 Address Bus

Carries memory location (address of data).

🎛️ Control Bus

Carries control signals like read/write commands.
